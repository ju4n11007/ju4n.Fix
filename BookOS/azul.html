Solucionar Pantalla Azul (BSOD)

Dificultad: Avanzada
Tiempo estimado: 30-45 minutos
1

Anota el código de error

Cuando aparezca la pantalla azul, anota el código de error. Suele aparecer en la parte inferior de la pantalla.

Ejemplo: STOP: 0x0000007B, IRQL_NOT_LESS_OR_EQUAL, etc.

Códigos comunes:

  • 0x0000007B: Problemas con el disco duro/controlador
  • 0x00000050: Problemas de memoria RAM
  • 0x000000D1: Problemas con controladores
2

Reinicia en modo seguro

El modo seguro carga Windows con controladores mínimos, útil para diagnosticar problemas.

Para Windows 10/11:

  1. Mantén presionado Shift mientras haces clic en "Reiniciar"
  2. Selecciona "Solucionar problemas" > "Opciones avanzadas"
  3. Haz clic en "Configuración de inicio"
  4. Presiona 4 o F4 para "Habilitar modo seguro"
  5. Espera a que Windows se inicie en modo seguro
3

Desinstala software reciente

Si el problema comenzó después de instalar un programa o controlador, desinstálalo.

  1. Abre "Configuración" > "Aplicaciones"
  2. Busca el software instalado recientemente
  3. Haz clic en él y selecciona "Desinstalar"
  4. Sigue las instrucciones para completar la desinstalación
  5. Reinicia tu PC y verifica si el problema persiste
4

Actualiza o reinstala controladores

Los controladores defectuosos son una causa común de pantallas azules.

  1. En modo seguro, abre "Administrador de dispositivos"
  2. Busca dispositivos con icono de advertencia (triángulo amarillo)
  3. Haz clic derecho en el dispositivo y selecciona "Actualizar controlador"
  4. Elige "Buscar automáticamente controladores"
  5. Si no funciona, descarga el controlador del sitio del fabricante
5

Ejecuta diagnóstico de memoria y disco

Problemas de hardware como RAM defectuosa pueden causar pantallas azules.

Diagnóstico de memoria RAM:

  1. Escribe "Diagnóstico de memoria de Windows" en la búsqueda
  2. Haz clic en "Reiniciar ahora y comprobar si hay problemas"
  3. Tu PC se reiniciará y ejecutará la prueba automáticamente
  4. Los resultados aparecerán después del reinicio

Comprobación de disco:

  1. Abre el símbolo del sistema como administrador
  2. Escribe: chkdsk C: /f /r y presiona Enter
  3. Presiona Y para programar la verificación en el próximo reinicio
  4. Reinicia tu PC y espera a que complete la verificación

Consejos Adicionales

Restaura el sistema

Si nada funciona, intenta restaurar Windows a un punto anterior cuando funcionaba correctamente. Ve a "Configuración" > "Actualización y seguridad" > "Recuperación".

Verifica temperatura del hardware

El sobrecalentamiento puede causar pantallas azules. Usa software como HWMonitor para verificar temperaturas de CPU y GPU.

Prueba la memoria RAM

Si sospechas de problemas con la RAM, prueba cada módulo individualmente. Retira todos menos uno y prueba si el error persiste.

¿Pantalla azul persistente?

Si el problema continúa después de seguir estos pasos, probablemente sea un problema de hardware que requiere atención profesional.
